# HXRMXS UNO Engine

A **pedagogical transition engine** that treats UNO as the training trace for how a teacher-state moves through a learner-state. The graph is Hermes; the LLM is only a renderer after the graph has selected the move.

## Core thesis

```
UNO is not answer training.
UNO is transition training.
It teaches: state → move → next state with metacognitive commentary
around why that move was chosen.
```

## How it works

```
user_text
→ classify (graph-owned labels)
→ retrieve (vector search over mythought_hyperedges)
→ aggregate (incidence counting)
→ select_pathway (graph-owned: weights + policy)
→ compose graph_mythought
→ select response form
→ render (template, no LLM)
→ validate
→ save trace
→ feedback updates policy weights
```

The critical architectural rule:

```
No LLM in cognition.
Optional LLM in rendering only.
```

`graph_mythought` is composed from the graph's selected pathway, not generated by an LLM.

## Pipeline modules

| Module | Role |
|---|---|
| `src/retrieve.py` | Vector search over mythought hyperedges |
| `src/pathway.py` | Aggregate incidences, score candidates with policy weights, select the pathway |
| `src/graph_mythought.py` | Compose the graph_mythought JSON from selected pathway + source hyperedges |
| `src/render.py` | Select response form, render from template, validate output |
| `src/graph.py` | LangGraph orchestration |
| `src/weights.py` | Save runs, apply feedback, update policy_weights |

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Build data

```bash
# Parse UNO transcripts into SQLite hyperedge store
python3 -m src.ingest_uno

# Embed my_thought text for vector retrieval
python3 -m src.embed
```

## Run

```bash
streamlit run src/app.py --server.port 8501
# or
python3 server.py  # HTTP server on port 2222
```

## Test

```bash
pytest tests/
```

## Key concepts

- **UNO**: Dataset of annotated therapy dialogues. Each assistant turn has a `[PEDAGOGY]` block with teaching metadata.
- **my_thought**: The teacher's internal monologue — the policy-rationale layer, not just decorative text.
- **HXRMXS**: The function taxonomy (UM/RM/SM/ME) + mechanism shapes + register system.
- **Hyperedge**: A my_thought bound to its learner state, behavior tags, mechanism, move, register, and traps.
